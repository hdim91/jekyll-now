The Linear Probability Model

Suppose the response variable is binary,

\[\begin{align} y = 1 \text{if an individual chooses to buy a house} \\ y = 0 \text{ if an individual chooses not to buy} \end{align}\]

The linear probability model has the general form of equation below:

\[\begin{align} y = E(y) + e = \beta_1 + \beta_2x_2 + ... + \beta_kx_k + e \end{align}\]

The Probit Model

Probit model: It assumes a nonlinear relationship between the response variable and regressors, this relationship seing the cumulative distribution function of the normal distribution

\[\begin{align} p = P[y=1] = E(y|x) = \Phi(\beta_1 + \beta_2x) \end{align}\]

The slope of the regression curve is not constant, but is given by the standard normal density function:

\[\begin{align} \frac{dp}{dx} = \phi(\beta_1 + \beta_2x)\beta_2 \end{align}\]

Predictions of the probability that y = 1 are given by:

\[\begin{align} \hat{p} = \Phi(\hat{\beta}_1 + \hat{\beta}_2x) \end{align}\]

The equation can be used to calculate partial effects of an increase in dtime by one unit (10 minutes):

xdtime <- data.frame(dtime=2)
predLinear <- predict(auto.probit,xdtime,
                      data=transport, type="link")
DpDdtime <- coef(auto.probit)[[2]]*dnorm(predLinear)
DpDdtime
##         1 
## 0.1036899

The function predict has the following general form: predict(object, newdata=NULL, type=c("link","response","terms"),sefit=FALSE,dispersion=NULL,terms=NULL,na.action=na.pass...)

The option argument newdata must be a data frame containing the new values of the regressors for which the prediction is desired.

xdtime <- data.frame(dtime=3)
predProbit <- predict(auto.probit,xdtime,
                      data=transport, type="response")
predProbit
##         1 
## 0.7982915

The Logit Model for Binary Choice

Below equations define expressions of the logit model (The two expressions are equivalent)

\[\begin{align} p = \Lambda(\beta_1 + \beta_2x) = \frac{1}{1+e^{-\beta_1+\beta_2x}} \\ p = \frac{exp(\beta_1 + \beta_2x)}{1+exp(\beta_1+\beta_2x)} \end{align}\] Following equation gives the marginal effect of a change in the regressor \(x_k\) on the probability that y = 1 \[\begin{align} \frac{\partial p}{\partial x_k} = \beta_k\Lambda(1-\Lambda) \end{align}\]

medGrades <- median(nels_small$grades)

newdat <- data.frame(grades = c(medGrades,fifthPercentileGrades))
pred <- predict(nels.multinom,newdat,"probs")
